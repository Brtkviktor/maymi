# 🌸 Maymi

> Plataforma Java para integração entre Minecraft, Discord e Web.

## Tecnologias

- Java 21
- Maven
- Paper API
- JDA
- Spring Boot (em breve)
- PostgreSQL (em breve)

## Arquitetura

```text
Discord
      │
      ▼
 Maymi Core
      │
 Socket TCP
      │
      ▼
Plugin Paper
      │
Minecraft
```

## Módulos

```
maymi-common
maymi-core
maymi-paper
```
